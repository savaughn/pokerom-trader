◆ moves

uint16_t pksav_gba_pokemon_attacks_block::moves[PKSAV_GBA_POKEMON_NUM_MOVES]

Indices of the Pokémon's moves.

If a move slot does not have a move, its value will be 0.

These values are stored in little-endian, so access and modify them with the function pksav_littleendian16.
