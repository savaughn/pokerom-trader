◆ PKSAV_VERSION_MAJOR

#define PKSAV_VERSION_MAJOR   1

PKSav major version.

This number is incremented with wide-reaching library changes. Two versions with differing major version numbers are virtually guaranteed to be incompatible.

◆ PKSAV_VERSION_MINOR

#define PKSAV_VERSION_MINOR   999

PKSav minor version.

This number is incremented with new feature support, usually the addition of a new generation of games or adding features to an existing component. Existing ABI may be compatible but is not guaranteed.

◆ PKSAV_VERSION_PATCH

#define PKSAV_VERSION_PATCH   999

PKSav patch version.

This number is incremented when a significant set of bugfixes is added without breaking ABI compatibility.
