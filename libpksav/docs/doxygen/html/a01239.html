uint8_t* pksav_gen2_pokemon_storage::p_current_box_num

The number of the current Pokémon box (0-11).

There are 12 Pokémon boxes, and this value (0-based) shows which one will be deposited and withdrawn from.

The ::PKSAV_GEN2_CURRENT_POKEMON_BOX_NUM_MASK should be used to access or set this value.

struct pksav_gen2_pokemon_box* pksav_gen2_pokemon_storage::p_current_box

A pointer to the current Pokémon box.

Pokémon are only deposited or withdrawn from this box, and its data is switched out when the current box is changed.
