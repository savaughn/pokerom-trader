#define PKSAV_GBA_POKEMON_ORIGIN_GAME_MASK   (((uint16_t)(0x780)))

The mask for determining a Pokémon's origin game.

To access the value, apply the mask to the pksav_gba_pokemon_misc_t.origin_info field and shift the result right by PKSAV_GBA_POKEMON_ORIGIN_GAME_OFFSET.

To modify the value, apply the inverse of the mask to the pksav_gba_pokemon_misc_t.origin_info field. Then shift the new value left by PKSAV_GBA_POKEMON_ORIGIN_GAME_OFFSET and perform a logical-or with this value and the field.

◆ PKSAV_GBA_POKEMON_OBEDIENCE_MASK

#define PKSAV_GBA_POKEMON_OBEDIENCE_MASK   ((uint32_t)(1 << 31))

The mask for determining whether a Mew or Deoxys will be obedient.

If this bit is not set, a Mew or Deoxys will not listen to the player, no matter how many badges he/she has. This field has no effect in any other Pokémon.

Apply this mask to the pksav_gba_pokemon_misc_t.ribbons_obedience field to modify this bit.
