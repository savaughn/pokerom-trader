enum pksav_error pksav_gen1_import_text ( const uint8_t *  p_input_buffer,
char *  p_output_text,
size_t  num_chars 
)

Convert a string from Generation I format to a UTF-8 C string.

In Generation I games, strings are stored with a proprietary character map.

Both buffers passed into the function must be at least the length passed in the num_chars parameter, or it will result in undefined behavior.

Parameters
input_bufferGeneration I string
output_textoutput buffer in which to place converted text
num_charsthe number of characters to convert
Returns
PKSAV_ERROR_NONE upon success
PKSAV_ERROR_NULL_POINTER if input_buffer or output_text is NULL
