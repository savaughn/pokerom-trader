◆ species

uint8_t pksav_gen2_pokemon_box::species[PKSAV_GEN2_BOX_NUM_POKEMON+1]

The species indices of the Pokémon in the box.

When the box is viewed in-game, it is this value that determines what Pokémon is showed, not the struct pksav_gen2_pc_pokemon.species value.

If this field is set to 0xFD, the Pokémon will be in an egg. Its field in struct pksav_gen2_pc_pokemon will correspond to its species when hatched.

The first index after the last Pokémon in the box should always be set to 0xFF.
