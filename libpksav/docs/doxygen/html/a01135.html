◆ species

uint8_t pksav_gen1_pokemon_box::species[PKSAV_GEN1_BOX_NUM_POKEMON+1]

The species indices of the Pokémon in the box.

When the box is viewed in-game, it is this value that determines what Pokémon is shown, not the pksav_gen1_pc_pokemon_t.species value.

The first index after the last Pokémon in the box should always be set to 0xFF.
