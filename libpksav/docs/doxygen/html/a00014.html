enum pksav_gba_contest_ribbon_level

What level of contest ribbon a given Pokémon has.

For each contest type, there is a 3-bit slot in the Pokémon's Hoenn ribbon field that stores which level of contest the Pokémon has one for that particular type. This enum represents all valid values.

Enumerator
